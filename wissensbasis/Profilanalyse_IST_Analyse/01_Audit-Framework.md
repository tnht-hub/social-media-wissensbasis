# Säule 1 - Audit-Framework: Den Auftritt strukturiert aufnehmen

Diese Säule liefert das Gerüst für die Bestandsaufnahme: erst alles zählen (Kanal-Inventar), dann prüfen, ob es zusammenpasst (Konsistenz), dann jedes Profil im Detail (Bio, Visual, Tonalität). Reihenfolge ist Absicht. Wer mit dem Detail beginnt, verliert das Gesamtbild.

Arbeite mit Checklisten und einer einzigen Inventar-Tabelle. Alles, was du findest, kommt hinein, auch tote und vergessene Profile.

---

## 1. Kanal-Inventar

Das Inventar ist die Grundlage für alles Weitere. Es beantwortet: Welche Profile existieren überhaupt, offiziell und inoffiziell, aktiv und tot.

**Vorgehen:** Suche den Markennamen auf jeder relevanten Plattform plus über Google. Frage die Kundin nach Zugängen und nach alten Accounts. Prüfe auch Schreibvarianten und Sub-Brands.

| Feld | Was eintragen |
|---|---|
| Plattform | Instagram, TikTok, LinkedIn, Facebook, X, YouTube, sonstige |
| Profilname / Handle | exakt wie angezeigt |
| URL | vollständig |
| Status | aktiv / inaktiv / verwaist / Fake-Verdacht |
| Letzter Post | Datum |
| Follower | Zahl mit Erhebungsdatum |
| Verifiziert | ja / nein |
| Verantwortlich | Person oder „unklar" |
| Zugang vorhanden | ja / nein |

**Checkliste Inventar:**
- [ ] Alle offiziellen Profile erfasst
- [ ] Nach verwaisten Alt-Profilen gesucht (alte Kampagnen, Standort-Seiten, frühere Namen)
- [ ] Nach Fake- oder Imitations-Profilen gesucht
- [ ] Inaktive Kanäle markiert (kein Post seit mehr als 90 Tagen)
- [ ] Zugangslage je Profil dokumentiert (oft der erste echte Quick Win)

---

## 2. Konsistenz-Check über Kanäle

Hier geht es nicht um ein einzelnes Profil, sondern um den Gesamteindruck: Würde eine Person, die die Marke auf Instagram kennt, sie auf LinkedIn wiedererkennen. Vier Dimensionen, je Plattform geprüft.

| Dimension | Geprüft wird | Bewertung |
|---|---|---|
| **Name / Handle** | Gleicher Handle überall? Auffindbar unter Markenname? | konsistent / abweichend |
| **Bio / Beschreibung** | Gleiche Kernaussage, gleicher Claim, gleiche Tonalität? | konsistent / abweichend |
| **Visual** | Gleiches Profilbild, gleiche Bildsprache, gleiche Farbwelt? | konsistent / abweichend |
| **Verlinkung** | Verweisen die Profile aufeinander und auf die Website? Stimmen die Links? | korrekt / fehlerhaft / fehlend |

**Checkliste Konsistenz:**
- [ ] Handle über alle Plattformen verglichen, Abweichungen notiert
- [ ] Profilbild identisch oder erkennbar verwandt
- [ ] Claim / Tagline in allen Bios gleich oder bewusst differenziert
- [ ] Website-Link in jeder Bio vorhanden und funktioniert (keine 404, kein altes Kampagnen-Linktree)
- [ ] Cross-Verlinkung der Kanäle untereinander vorhanden

Einheitliche Handles über alle Profile sind Standard-Empfehlung, weil sie die Auffindbarkeit erhöhen (Sprout Social, Asana). Abweichungen sind ein klassischer, billig behebbarer Befund.

---

## 3. Profil- und Bio-Optimierungs-Check

Jedes aktive Profil wird einzeln auf seine Funktionselemente geprüft. Die Bio ist die teuerste Zeile des Auftritts, sie entscheidet in Sekunden über Follow oder Weiterscrollen.

**Checkliste je Profil:**
- [ ] **Name-Feld** nutzt relevante Keywords (nicht nur den Markennamen), wo die Plattform es durchsucht (Instagram, TikTok)
- [ ] **Bio** sagt in der ersten Zeile, was die Marke ist und für wen
- [ ] **Wertversprechen** erkennbar, nicht nur Floskeln
- [ ] **Call-to-Action** vorhanden (Link, Aktion, nächster Schritt)
- [ ] **Link** aktuell und auf das richtige Ziel gerichtet
- [ ] **Kontaktmöglichkeiten** vollständig (je nach Plattform: Standort, E-Mail, Button)
- [ ] **Highlights / Pinned Posts** gepflegt und aktuell (Instagram-Highlights, LinkedIn-Featured, X-Pinned)
- [ ] **Kategorie / Branche** korrekt gesetzt
- [ ] **Vollständigkeit** des Profils nach Plattform-Standard (fehlende Felder notieren)

Tabelle zur Bewertung je Profil:

| Profil | Bio-Klarheit (1 bis 5) | CTA vorhanden | Link korrekt | Highlights gepflegt | Quick-Win-Notiz |
|---|---|---|---|---|---|

---

## 4. Visual-Check

Der visuelle Eindruck entscheidet über Wiedererkennung. Hier wird beschrieben, nicht umgestaltet.

**Checkliste Visual:**
- [ ] **Profilbild** in korrekter Auflösung, nicht beschnitten, plattformgerecht
- [ ] **Titelbild / Banner** vorhanden, aktuell, richtig formatiert (Facebook-Header, LinkedIn-Banner, YouTube-Channel-Art, X-Header)
- [ ] **Feed-Gesamteindruck** kohärent (Farbwelt, Bildlogik, Wiederkehrendes)
- [ ] **Templates** erkennbar oder Wildwuchs
- [ ] **Logo-Verwendung** einheitlich
- [ ] **Lesbarkeit** von Text in Grafiken auf Mobil
- [ ] **Format-Disziplin** (werden Plattform-Formate respektiert, Hochformat für Reels und Co.)

Notiere je Plattform einen Gesamteindruck in einem Satz plus drei konkrete visuelle Auffälligkeiten. Keine Geschmacksurteile, sondern beschreibbare Beobachtungen („wechselnde Schriftarten über die letzten 9 Posts", nicht „sieht unprofessionell aus").

---

## 5. Tonalitäts-Check

Hier wird die Sprache des Auftritts beschrieben, nicht bewertet, ob sie „gut" ist. Das ist ein IST-Befund, das Soll kommt in der Positionierung.

**Vorgehen:** Lies die letzten 15 bis 20 Captions je aktivem Hauptkanal. Notiere Muster.

| Dimension | Beobachtung |
|---|---|
| Ansprache | Du / Sie / neutral / wechselnd |
| Register | locker / sachlich / werblich / fachlich |
| Satzlänge | kurz / lang / gemischt |
| Emoji-Nutzung | keine / sparsam / viel |
| Hashtag-Praxis | Anzahl, Strategie erkennbar / Salat |
| Wiederkehrende Formeln | Begrüßungen, CTAs, Floskeln |
| Konsistenz über Kanäle | gleiche Stimme oder verschiedene |

**Checkliste Tonalität:**
- [ ] Ansprache über alle Posts konsistent
- [ ] Erkennbare eigene Stimme oder austauschbar
- [ ] CTA-Muster identifiziert (oft generisch und wiederholt)
- [ ] Abweichungen zwischen Plattformen dokumentiert (bewusst differenziert oder zufällig)

---

## Output dieser Säule

Am Ende von Säule 1 liegen vor:
1. Eine **Inventar-Tabelle** mit allen Profilen.
2. Eine **Konsistenz-Matrix** (Plattform x vier Dimensionen).
3. Pro aktivem Profil ein **ausgefüllter Profil-, Visual- und Tonalitäts-Check**.

Diese drei Artefakte fließen unverändert in die Scorecard (Säule 04) ein. Performance-Zahlen kommen in Säule 02 dazu.
