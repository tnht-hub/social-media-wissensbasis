# TikTok Analytics & Metriken: IST-Zustand Mai 2026

## Verfügbare Metriken (TikTok Studio)

### Video-Performance-Metriken

| Metrik | Bedeutung |
|---|---|
| **Views** | Gesamtzahl der Aufrufe |
| **Avg. Watch Time** | Durchschnittliche Sehdauer pro Zuschauer |
| **Completion Rate** | % der Zuschauer, die das Video zu Ende gesehen haben |
| **Replay Rate** | Wie oft das Video mehrfach angeschaut wurde |
| **Likes** | Anzahl der positiven Reaktionen |
| **Kommentare** | Anzahl der Kommentare |
| **Shares** | Weiterleitungen an andere Nutzer |
| **Saves** | In "Favoriten" gespeichert |
| **Profile Views from Post** | Wie viele Profilbesuche dieses Video ausgelöst hat |
| **New Followers from Post** | Follower, die durch diesen Post gewonnen wurden |
| **Reached Audience** | Einzigartige Nutzer, die das Video gesehen haben |
| **Traffic Source** | Woher kamen die Views: FYP, Following, Suche, Profil, etc. |

### Traffic-Quellen (wichtig für Strategie)

| Quelle | Bedeutung |
|---|---|
| For You | Algorithmisch verteilter Content |
| Following | Nur an Follower ausgespielt |
| Suche | Video wurde über TikTok-Suche gefunden |
| Profil | Direkt vom Profil aus aufgerufen |
| Sound-Seite | Über den Soundtrack gefunden |
| Hashtag-Seite | Über Hashtag entdeckt |

**Analyse-Tipp:** Ein hoher Anteil an Search-Traffic zeigt, dass der Content als Evergreen-Content funktioniert.

## Search Value (Creator Rewards Metrik)

Seit März 2024 relevante Metrik für Creator im **Creator Rewards Program**:
- Misst, wie gut ein Video zu aktiven Suchanfragen der Nutzer passt
- Höherer Search Value = höhere Monetarisierungsauszahlung
- Optimierbar durch: Creator Search Insights nutzen, Keywords in Caption, relevante Hashtags

## Engagement-Rate: Benchmarks 2026

| Engagement-Rate | Bewertung |
|---|---|
| < 1% | Unter Durchschnitt |
| 1 bis 3% | Durchschnittlich |
| 3 bis 6% | Gut |
| 6 bis 10% | Sehr gut |
| > 10% | Ausgezeichnet |

**TikTok vs. andere Plattformen:** TikTok hat historisch die höchsten organischen Engagement-Raten aller Plattformen. Durch zunehmende Content Saturation (Metricool: 31% Rückgang bei Views in 2026) sind diese Werte rückläufig, Nischen-Content hält jedoch höhere Raten.

## Welche Metriken wirklich wichtig sind

Nach Algorithmus-Relevanz gereiht:

1. **Avg. Watch Time & Completion Rate**, Wichtigste Kern-Metrik; bestimmt algorithmische Weiterverteilung
2. **Replay Rate**, Zeigt, dass Content gut genug für nochmaliges Anschauen ist
3. **Shares**, Höchste Reichweiten-Auswirkung
4. **Comments**, Stärkstes Engagement-Signal auf TikTok (mehr als Likes)
5. **Saves**, Evergreen-Signal
6. **Likes**, Direktes, aber geringer gewichtetes Signal
7. **Profile Views from Post**, Zeigt, ob Content Follower-Interesse weckt

## Analyse-Toolkit

### TikTok Studio Analytics
- **7-Tage / 28-Tage / 60-Tage-Vergleiche**
- Beste Post-Zeiten basierend auf Follower-Aktivität
- Traffic-Source-Analyse pro Video
- Top-Performers identifizieren

### Creator Search Insights
- Zeigt, welche Keywords und Themen aktuell in TikTok gesucht werden
- Gegliedert nach Content-Gap (Themen, die wenig Content haben, aber oft gesucht werden)
- Ideal für Keyword-optimierten Content

---

## Wöchentliche Analytics-Review-Routine

1. **Top-3-Videos** der Woche identifizieren: Was haben sie gemeinsam? (Hook, Format, Sound, Länge, Thema)
2. **Completion Rate analysieren:** Wo brechen Zuschauer ab? → Hook oder Pacing verbessern
3. **Traffic-Sources:** Wie viel kommt von FYP vs. Search vs. Following?
4. **Search Value beobachten** (Creator Rewards Teilnehmer)
5. **Follower-Wachstum:** Welche Posts bringen neue Follower?
6. **Creator Search Insights:** Welche Themen haben hohes Suchvolumen + niedrigen Content-Bestand?
